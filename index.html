Biomedical Image DeepLearning

딥러닝 수학 기초에 대해 공부했습니다. 미적분학 지식이 어떻게 딥러닝에서 응용되는지 알게 되었고, 이후 "확률과 통계" 및 "선형대수학"을 더 학습하는 계기가 되었습니다.

Lung Segmentation U-net 모델을 활용하여 bladder image segmentation이 가능한 모델로 수정하였습니다.

기존 모델은 binary input 이미지로 학습하여 gray scale로 출력하는 형태입니다.
먼저 해당 부분을 binary input & binary output으로 변경했습니다.

데이터 Augmentation을 하는 python 코드를 작성했습니다. Image Flipping, Cropping 등의 방법을 구현하였습니다.

U-net 모델의 구조

U-net은 자체적으로 데이터 증강(Data Augmentation)을 하는 모델입니다.
그리하여 비교적 이미지 수가 적은 의료 이미지 딥러닝에 활용될 수 있습니다.

모델을 수정할 때 고려해야했던 것은 Lung 이미지와 bladder 이미지의 크기 차이로 인한 accuracy 척도의 변화입니다. Lung 이미지는 1로 표시되는 영역이 비교적 넓지만, bladder 이미지는 1로 표시되는 영역이 넓어서 accuracy 값이 99 이상이 되어야 잘 학습된 결과를 얻을 수 있었습니다.

UGRP

오디오 데이터 피처를 이용한 샴 네트워크 딥러닝 모델과 유사도 측정

MFCC, Chromagram, Spectrogram 등의 오디오 피처(Feature)를 기반으로 Siamese network 인공지능을 이용해 음악 오디오의 유사도를 측정하는 연구를 진행했습니다.

음악 과학 기술에 크게 관심을 가지게 된 계기입니다.
한국음악지각인지학회(KSMPC) 참여와 연구실 선배와의 미팅을 통해 음악 기술 분야의 다양한 연구를 접한 후 "내가 제일 잘 할 수 있는 분야", "내가 제일 하고 싶은 공부"라 느꼈습니다.

1년 동안 팀장으로서 회의 조율과 서류 처리를 담당하였습니다.
꾸준히 연구하면서 능동적으로 자료를 찾고 생각하는 능력을 기르고 연구 시 마주할 수 있는 여러 시행착오들을 경험해보는 시간이었습니다.

인공지능 모델 구축

음악 연구에 주로 사용되는 피처들과 딥러닝 기초에 대해 학습했습니다. Dual Input CNN 및 Regression 모델을 오디오 유사도 측정에 적용할 수 있을지 고민해보았으며,
딥러닝 모델을 고안하면서, 심장 신호를 처리하는 논문에서 아이디어를 얻기도 하였습니다.
이때 분야의 융복합의 중요성을 깨닫고 이후 뇌 신호처리 분야의 연구실에서 인턴을 하는 계기가 되었습니다.

두 오디오의 유클리디안 거리를 도출하는 Siamese network 모델을 구축하였고, 잡음 등으로 Augmentation된 데이터로 학습시켰습니다.
학습시킨 각 피처별로 결과의 경향성이 달라 일반인 20명을 대상으로 설문조사를 진행했고, 설문 결과와 같은 경향을 보이는 MFCC 데이터셋을 최종적으로 선택했습니다.

QBOL Lab

Quantum & Biomedical Optics Lab

(현재) 나선 악보(Spiral Music Sheet)의 3D 시각화에 대해 연구하고 있습니다.

나선 악보는 나선 위에 음이 표현되는 방식입니다.

나선형 악보 위에서 표현되는 음은 주파수가 높아질수록 나선을 따라 시계방향으로 진행합니다.
화음은 각 음들을 나타내는 점을 이으면 삼각형이나 사각형 등의 도형으로 보이므로 화음의 특징을 직관적으로 확인할 수 있습니다.

Imaj7 V7 IIm7

기존에 2D 나선 악보를 z방향으로 확장하여 시간에 따라 +z 방향으로 악보가 진행되도록 Matlab으로 시각화했습니다.


MIDI & Spiral Sheet

matlab으로 MIDI파일을 불러와 3D 나선 악보로 그립니다.
note의 on off 정보를 이용합니다. 하나의 note가 on된 이후 off될 때까지 z방향으로 그래프가 그려집니다.

Example : C major Scale

흰 건반을 흰색으로, 검은 건반을 검정 면으로 표현하였습니다.
시간이 지남에 따라 z=t 평면이 양의 방향으로 움직입니다.

3D 나선악보와 2D 나선 악보를 같이 시각화하여 현재 시각에 어떤 음이 재생되고 있는지를 확인할 수 있고, 3D로 이후의 진행을 예측할 수 있습니다.
